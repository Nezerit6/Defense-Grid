# Defense Grid

A mod for [Mindustry](https://github.com/Anuken/Mindustry) for the Serpulo adding new turrets to vanilla gameplay

[![Stars](https://img.shields.io/github/stars/Nezerit6/Defense-Grid?color=yellow&label=⭐%20Please%20Star%20Mod!&style=for-the-badge)](https://github.com/Nezerit6/Defense-Grid)
[![Version](https://img.shields.io/github/v/release/Nezerit6/Defense-Grid?color=green&include_prereleases&label=Latest%20Version&logo=github&style=for-the-badge)](https://github.com/Nezerit6/Defense-Grid/releases)
[![Downloads](https://img.shields.io/github/downloads/Nezerit6/Defense-Grid/total?color=red&label=Total%20Downloads&logo=github&style=for-the-badge)](https://github.com/Nezerit6/Defense-Grid/releases)

### Adds:
- Arclet (after Arc) — Electric turret
- Needler (after Scatter) — Rapid-fire ammo-based turret
- Shatter (after Salvo) — Fragmenting artillery
- Cryolance (after Lancer) — Charge-based cryofluid launcher
- Arcflash (after Salvo) — Guided missile turret
